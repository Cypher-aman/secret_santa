## Getting Started

### Prerequisites

- Node.js installed
- A Supabase account and project

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```
3. Set up environment variables:
   Create a `.env` file in the root directory:
   ```
   REACT_APP_SUPABASE_URL=your_project_url
   REACT_APP_SUPABASE_ANON_KEY=your_anon_key
   ```

### Database Setup

Run the following SQL in your Supabase SQL Editor:

```sql
create table participants (
  id bigint generated by default as identity primary key,
  name text not null,
  has_drawn boolean default false,
  is_picked boolean default false,
  picked_who text,
  created_at timestamp with time zone default timezone('utc'::text, now())
);

-- Enable Row Level Security
alter table participants enable row level security;

-- Add policy for access
create policy "Enable all access" 
on participants for all 
using (true) 
with check (true);
```

### Running the App

```bash
npm start
```

## How to Play

1. **Admin Setup**: 
   - Go to the Admin Panel (Password: `santa123`)
   - Add all participants
   
2. **The Game**:
   - Participants visit the home page
   - Click their name
   - Shuffle the cards
   - Pick a card to reveal their Secret Santa target!
   - Save the ticket image
   
3. **Reset**:
   - The Admin can reset draws or clear all data from the panel.
