## Database Setup

Run the following SQL in your Supabase SQL Editor to create the necessary table:

```sql
create table participants (
  id bigint generated by default as identity primary key,
  name text not null,
  has_drawn boolean default false,
  is_picked boolean default false,
  picked_who text,
  created_at timestamp with time zone default timezone('utc'::text, now())
);
```

## Security Policies (RLS)

Enable Row Level Security (RLS) and add these policies to allow access:

1. Enable RLS on the `participants` table.
2. Add a policy for full access (since this is a simple game app):
   - Name: `Enable all access`
   - Target roles: `anon`, `authenticated`
   - Using expression: `true`
   - With check expression: `true`

Or run this SQL:

```sql
alter table participants enable row level security;

create policy "Enable all access"
on participants
for all
using (true)
with check (true);
```
